## 功能

1. js、css、html、images的压缩

2. 浏览器缓存问题，添加hash

3. 启动server，并浏览器自动刷新


## 项目的安装与启动

**1、检查node、npm、npx是否已安装并正确安装**

**2、全局安装gulp-cli**

​	`npm install --global gulp-cli`

**3、安装package.json下的所有插件**
 
 `npm install`

**5、压缩图片, 执行gulp命令**

​	`gulp img`

**5、项目启动， 执行gulp命令**

 `gulp`

​**6、打包成zip格式**

 `gulp zip`
